# 4-in-a-Row for CloudPhone

A modern Connect Four–style game built for CloudPhone devices (240×320 resolution).  
Features include:
- **Game Modes**: Vs Player (local pass‑and‑play) and Vs Computer (basic AI).  
- **Difficulty Levels**: Easy, Medium.  
- **Controls**: T9 keypad support (2,4,6,8 arrows and 5 to select).  
- **Material Design UI**: Clean, responsive layout with About and Help pages.

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/4-in-a-Row`  
4. Upload icons if required.  
5. Save and refresh — **4-in-a-Row** will appear on your CloudPhone.
